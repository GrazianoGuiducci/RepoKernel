# Memory Delta

Preserve only useful deltas.

## Preserve

```text
active_surface:
current_next:
accepted_correction:
source_of_truth:
verified_state:
boundary:
residue_not_to_follow:
first_safe_action:
durable_decision:
reusable_rule:
source_promotion_or_deprecation:
claim_to_verify:
```

## Do Not Preserve

```text
ordinary_chat:
duplicate_summary:
temporary_speculation:
credentials:
private_material_in_public_surfaces:
```
