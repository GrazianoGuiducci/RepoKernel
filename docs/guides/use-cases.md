# RepoKernel Use Cases

## AI-Assisted Software Repository

Preserve current state, source authority, test expectations, safe next action
and memory deltas across AI sessions.

## AI/RAG Or Model Project

Track datasets, model assumptions, evaluation plans, failure memory and next
experiments without mixing them with code-only README content.

## Editorial Or Book Project

Preserve corpus sources, chapter state, editorial decisions, style guides, open
questions and next writing packet.

## Business Operating Function

Preserve contacts, offers, decision capsules, evidence, follow-up boundaries
and sustainability routes.

## Research/Lab Project

Separate claims, falsifiers, evidence, reports, rejected hypotheses and
repeatable cycles.

## External Collaborator Repo

Use A1 observe-and-propose mode. Inspect, propose a setup capsule and stop
before any write.
