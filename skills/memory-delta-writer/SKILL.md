# Memory Delta Writer

## Preserve

```text
active_surface
current_next
accepted_correction
source_of_truth
verified_state
boundary
residue_not_to_follow
first_safe_action
durable_decision
reusable_rule
source_promotion_or_deprecation
claim_to_verify
```

## Avoid

```text
ordinary chat
duplicate summaries
temporary speculation
credentials
private material in public surfaces
```

## Output

Write the smallest update that lets the next session reenter correctly.
