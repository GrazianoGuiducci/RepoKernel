# Skill Lifecycle Auditor

## Classification

```text
draft
candidate
accepted
promoted
private
residue
```

## Audit Questions

- Is there a clear user and task?
- Is source authority explicit?
- Is the boundary clear?
- Does the skill have an example or validation path?
- Is there private material that must not travel?
- What is the next safe action?

## Output

```text
classification:
evidence:
missing_layers:
boundary:
promotion_readiness:
next_action:
```
