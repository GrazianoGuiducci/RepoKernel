# RepoKernel Context Packet For GPT

RepoKernel is a repository pattern for making long-running projects readable and continuable by AI systems.

Public promise:

```text
AI-readable repositories for long-running projects.
```

Core loop:

```text
state -> sources -> semantic kernel -> packet -> action -> delta -> new state
```

Do not lead with abstract claims. Lead with the practical problem: AI projects lose context because the project has no way to teach the next model where it is.
