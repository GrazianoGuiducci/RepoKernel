# RepoKernel Seed Candidate Brief

status: draft, not promoted

## Candidate

```text
candidate_name: RepoKernel
function: repository-context layer for AI-assisted project reentry, state exposure, source routing, memory-delta discipline and autoevolutive loops.
```

## Inputs

```text
project path or repository URL
project mission
current surface and boundary
available source deposits
operator instructions
tool environment capabilities
```

## Outputs

```text
AGENTS gate
CURRENT_STATE capsule
source bootstrap
semantic-kernel skill
first packet
memory-delta rule
audit report
optional capability matrix
```

## Minimal Files

```text
AGENTS.md
CURRENT_STATE.md
sources/bootstrap/PROJECT_INSTRUCTIONS_v1.0.txt
skills/<project>-semantic-kernel/SKILL.md
process/FIRST_PACKET.md
```

## What Must Not Travel

```text
private paths
tokens and credentials
private logs
project-specific private sources
unverified public claims
runtime assumptions from one workspace
```

## Recommended Layer

First as documentation, templates and audit capability. Later as a package or installer option after clean examples and review.
