## Result

RepoKernel now has a public-safe procedure for the first external-style A1
pilot.

The purpose is to observe a public or authorized repository and produce a
bounded Project Kernel proposal without writing to the target repository.

## Why This Exists

RepoKernel is a Project Kernel compiler, not only a local scaffold. Its first
external proof should show that it can:

```text
resolve target identity;
read direct repository evidence;
classify the project maturity;
identify the lowest useful L0/L1/L2 layer;
propose files or state structures;
preserve a no-write proof;
protect private relationship context.
```

## Public / Private Boundary

The pilot packet deliberately separates:

```text
RepoKernel:
  technical method, sanitized capsule, no-write evidence.

Business/contact surface:
  person-specific context, call notes, relationship status, follow-up.
```

This prevents a public RepoKernel repository from becoming a contact notebook.

## Current Authority

Allowed:

```text
read public or authorized sources;
inventory repository metadata;
produce a setup capsule;
draft a proposal;
record no-write evidence.
```

Blocked:

```text
apply;
write target files;
open PRs, issues or discussions;
publish contact details;
promote to Seed;
claim full integration into an external project.
```

## Pilot Shape

The first real candidate should be handled as:

```text
A1 observe-and-propose;
single selected repo or workflow;
lowest useful layer only;
technical proposal before implementation;
operator approval before any external action.
```

## Reusable Output

The reusable output is not a finished installer. It is:

```text
identity and source check;
repository observation capsule;
recommended RepoKernel layer;
proposed files;
what not to touch;
next bounded action;
stop rule.
```

## Readiness

This report does not unblock public installer claims. It only makes the A1
external-style pilot procedurally ready.

The RepoKernel Phase 1 boundary remains:

```text
validate -> inspect -> plan -> guides -> audit -> review packet
no apply command
no external write
```
