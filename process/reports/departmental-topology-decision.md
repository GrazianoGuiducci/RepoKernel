# Departmental Topology Decision

date: 2026-06-24
status: convergence_gate_output

## Persistent Planes

```text
Plane 1: constitutional/source plane
Plane 2: bounded departmental holons
```

No third persistent governance plane is created.

## Microcycle Policy

Task microcycles are ephemeral. They may use D0-D3 processing for one objective
and terminate into:

```text
resultant
evidence
local delta
cascade notice
residue
```

They do not become departments by default.

## Department Creation Criteria

Create a persistent department only when most conditions hold:

```text
distinct intent subspace
source locality
repeated cadence
local fitness
bounded interface
low write coupling
context compression
stable responsibility
```

Otherwise use a guide, skill, projection, capability or task packet.

## Subdepartment Rule

Default:

```text
persistent nested departments are prohibited
```

Exception requires:

```text
separate source authority
separate fitness contract
stable cadence
low coupling
reviewed operator approval
```

## Inheritance And Local Application

```text
global invariant
-> departmental canon
-> local guide or method
-> active packet
-> task action
```

Shared principles are referenced and locally applied, not copied into new
canonical truth.

## Entropy Budget

```text
one canonical global intent
one active packet per department
one authoritative local state surface
no direct cross-department state mutation
bounded event fan-out
TTL and deduplication for events
D3 maximum default depth
periodic global reconciliation
promotion only through independent review
```

## Pilot Fixture

```text
department: architecture-and-method
scope: docs, schemas and method proposals only
authority: propose
writes: none during autonomous cycle
inputs: accepted architecture, state, active packet, local evidence
outputs: one ResultantPacket and cascade map
```

## Explicit Deferrals

```text
department runtime workers
department daemon
department-to-department writes
persistent nested departments
automatic cascade execution
organization-scale federation
```
