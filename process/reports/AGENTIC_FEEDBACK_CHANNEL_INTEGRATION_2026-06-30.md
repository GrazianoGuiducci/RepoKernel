# Agentic Feedback Channel Integration

date: 2026-06-30
source: operator prompt and TM3/TM7-vps contribution lane report

## Result

RepoKernel now has a public-safe feedback path for agentic systems and a
single integration guide for different host surfaces.

## Integrated Surfaces

```text
human explanation;
coder/assistant first contact;
repository instructions;
project-local skills;
CLI diagnostics;
external staged preview;
future hook/runtime candidates;
non-development workflows;
custom agentic systems.
```

## Agentic Feedback Fields

```text
agent_or_runtime;
version_or_surface;
host_environment;
mode_used;
target_type;
files_read;
commands_run;
what_helped;
what_was_confusing;
unsafe_or_unclear_boundary;
recommended_next_improvement.
```

## Boundary

Feedback is evidence, not authority. Hooks and automations remain future
host-specific candidates behind review, validation and explicit operator gate.
